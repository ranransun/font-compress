1. 解决了什么问题
	- 下载的字体非常大 一般都有十几兆，这个压缩之后只有几kb大小(例如[字体天下](https://www.fonts.net.cn/font-35942188604.html)下载的字体)
2. 主要是用`font-spider`插件 ♥️ 在此感谢该插件作者 ♥️
3. 使用方式
	- `git clone`该项目到本地
	- 执行`npm install`
	- 把想要使用的字体文件放到目录下，然后在html文件第8行引入使用
	- 执行`npm run compress`
	- 字体源文件即被压缩为几kb大小供我们使用，.font文件夹下是原始大小的字体文件
	- 可以打开html文件预览字体效果
